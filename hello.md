day 1  
- 二进制中1的个数  
常规解法  
1、将1 每次 <<1 与 x 进行&操作 记录结果为1的次数   
2、怎么确定 <<1 结束条件 <<1的数值大于x  
科学解法  
1、原数减去1之后最右边的1都会变成0 ， 用减1后的数和原数进行&操作  
    ```
    class Solution {
    public:
        int  NumberOf1(int n) {
            //常规解法
            int count = 0;
            int unsigned flag = 1;
            while(flag){
                
                if(n & flag)
                    count ++;
                flag = flag << 1;
            }
            
            //科学解法
            int count = 0;
            while(n){
                
                count ++;
                n = n & (n-1);
            }

            return count;
        }
    };
    ```
